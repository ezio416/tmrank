# TMRank

Track completion of maps within certain styles. Score points based on medals obtained and climb the leaderboard. Spl1nes website (https://tmrank.jingga.app) used for the data.

Scores/Rankings currently updated every 24 hours.

Originally created by [DodgeTheDuck](https://openplanet.dev/u/DodgeTheDuck), as of February 2026 this plugin is now being maintained by [Ezio](https://openplanet.dev/u/ezio416).
